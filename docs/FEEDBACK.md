# 📝 프로젝트 개선 및 피드백 노트

## 🎨 UI/UX 디자인 및 레이아웃
1. **화면 밝기 및 테마 개선**
   - 현재 화면이 너무 밝아 눈의 피로도가 높음.
   - 다크 모드 가시성 개선 또는 전체적인 배경 톤 다운 필요.

2. **공통 헤더 고정 (Sticky Header)**
   - 스크롤 시에도 홈, 로그인/로그아웃, 사운드, 다크모드 제어 버튼이 항상 상단에 표시되도록 고정.

3. **레이아웃 안정화**
   - 페이지 전환 시 레이아웃이 틀어지거나 깜빡이지 않도록 공통 레이아웃 구조 강화.
   - 로그인/로그아웃 버튼이 간헐적으로 사라지는 현상 수정.

4. **랜딩 페이지 고도화**
   - 사용자 유입을 위한 매력적인 랜딩 페이지(홈 화면) 디자인 업데이트 필요.

5. **에셋 퀄리티 업그레이드**
   - 앱 내 사용되는 효과음(Sound) 및 아이콘(Icon) 리소스 교체.

## 🎮 퀴즈 플레이 경험 (Interaction)
6. **정답/오답 피드백 시간 조절**
   - 현재 해설 화면이 너무 빠르게 넘어감.
   - **개선안**: '자동 넘김' On/Off 옵션 추가 또는 '화면 터치 시 다음 문제로 이동' 방식으로 변경.

## ✨ 기능 추가 및 확장 (Features)
7. **퀴즈 이어풀기 및 히스토리**
   - 중간에 종료한 퀴즈를 다시 이어서 풀 수 있는 기능 구현.
   - 과거 풀이 기록 열람 기능.

8. **나만의 퀴즈 관리 (My Quiz)**
   - 생성한 퀴즈를 관리(저장, 삭제, 다시 풀기)할 수 있는 기능.
   - 퀴즈 묶음(이전 Pool 시스템)을 DB에 저장하여 체계적인 복습 지원.

9. **PDF 기반 퀴즈 생성 (Backlog)**
   - PDF 문서를 업로드하여 퀴즈를 생성하는 기능.
   - 우선순위 낮음 (추후 개발 또는 외부 무료 서비스 링크 연동 고려).

   10. **로딩 상태 시각화 및 UX 개선**
       - 퀴즈 로딩 시 진행 상황을 명확히 인지할 수 있도록 로딩 인디케이터(Indicator) 시인성 강화.
       - 다중 로딩 상태 발생 시 각 컨텍스트를 구분할 수 있도록 UI 개선 및 무한 로딩 이슈 해결.

   11. **페이지 네비게이션 흐름(Navigation Flow) 교정**
       - '내 퀴즈 다시 풀기' 중단 시 홈 화면이 아닌 '내 퀴즈 목록'으로 복귀하도록 라우팅 로직 수정. 
   12. **유사 정답 매칭 시스템 (Fuzzy Matching)**
       - 한글/영어 답변에 정규식 및 유사도 검사(90% 이상 일치)를 적용하여 정답 처리 유연성 확보.
       - 단순 띄어쓰기 및 문장 부호 차이는 오차 범위 내로 간주하여 정답으로 인정.
       - 유사 정답 처리 시, 오타 안내와 함께 원본 정답 및 해설을 표시하여 학습 효과 증대.
